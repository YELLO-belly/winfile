Disk Menu Commands

Contains the following commands.

Copy Disk

Copies the contents of a floppy disk onto another floppy disk. Use this command to make a backup copy of a floppy disk.

Label Disk

Assigns or changes the volume label for a floppy or hard disk.

Format Drive

Formats a drive.

Connect Network Drive

Connects your computer to a network drive. Depending on your network, this command may not be available.

Disconnect Network Drive

Disconnects your computer from a network drive. Depending on your network, this command may not be available.

Share As

Opens the Network access window to set file sharing permissions.

Select Drive

Enables you to change drives.
