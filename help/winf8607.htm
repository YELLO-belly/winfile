Using the Select Files Command to Select Files

To select files by using the Select Files command

1. From the File menu, choose Select Files.

2. In the Select Files dialog box, type the name of the file you want to select, and then choose the Select button. You can use wildcards to select a group of files; for example, *.TXT. Repeat this step if you want to select more files.

You can choose the Deselect button at any time to cancel a selection. The Deselect button cancels the selected files that you specify in the Files box.

3. Choose the Close button when you have selected all the files you want.
