Formatting a Floppy Disk

Before you can use a disk, you must format it. Formatting prepares the disk for use with your computer and deletes any previous information stored on it.

To format a floppy disk

1. Insert a floppy disk in a drive.

2. From the Disk menu, choose Format Drive.

3. In the Format Disk dialog box, specify the drive letter, disk capacity, volume label, and other options.

4. Choose the OK button.

Note:Before you format a floppy disk, a confirmation message appears. If you don't want to receive confirmation messages, you can turn them off by using the Confirmation command on the Options menu.
