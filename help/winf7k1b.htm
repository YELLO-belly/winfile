Selecting Files

Before you can work with a file, you must select it. There are several ways to select files in File Manager. If the files you want are visible in the directory window, you can use the mouse or keyboard to quickly select them. If the files are not visible, or if you want to select a group of files with similar names, use the Select Files command.
