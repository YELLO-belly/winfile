Searching for Files

To find a file or group of files, you can use the Search command. If files are found, they are listed in the Search Results window. You can select items in this window and perform such tasks as copying, deleting, moving, and printing files. The contents of the Search Results window change when you perform a new search.

To search for files or directories

1. Select the directory from which you want to start the search.

2. From the File menu, choose Search.

3. In the Search dialog box, specify what you want to search for and whether you want to search in subdirectories.

4. Choose the OK button.

To cancel the search, press ESC.
