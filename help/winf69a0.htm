Printing Files

You can print document files by using File Manager. A document file is associated with an application and represented by a document-file icon in the directory window.

To print a file by using a mouse

Drag the document-file icon to the minimized Print Manager application icon (Print Manager must be running).

To print a file by using the Print command

1. Select the document file you want to print.

2. From the File menu, choose Print.

3. The Print dialog box displays the selected file. If the file is not the one you want to print, specify a different file.

4. Choose the OK button.
