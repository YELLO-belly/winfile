Options Menu Commands

Contains the following commands.

Confirmation

Specifies whether confirmation messages are displayed.

Font

Changes the font used to display information in all directory windows.

Customize Toolbar

You can add, remove, and rearrange the buttons on the toolbar. Every command available on File Manager menus has an equivalent button.

Toolbar

Show/hide the toolbar.

Drivebar

Show/hide the drivebar.

Status Bar

Specifies whether the status bar is displayed at the bottom of the File Manager window.

Open New Window On Connect

When you connect to a network drive, File Manager opens a new window in which to display the directory information.

Minimize on Use

Reduces File Manager to an icon whenever you start an application. A check mark beside the command means it is in effect.

Save Settings on Exit

Saves the positions of the open directory windows whenever you quit File Manager. A check mark beside the command means it is in effect.
